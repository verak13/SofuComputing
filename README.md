# Soft computing projekat - Prepoznavanje pesme na osnovu isečka u igri SongPop2 <br />
Za korištenje programa potrebno je instalirati igricu [SongPop2](https://www.microsoft.com/en-us/p/songpop-2/9phcwq1s144x#activetab=pivot:overviewtab) i kreirati novi nalog. Takođe potrebno je uključiti Stereo Mix na računaru. <br />
Za pokretanje potrebno je pokrenuti igricu i pokrenuti modul songpop2.py.
